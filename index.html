Why does this website exit?

This summer I worked as a producer on an album of new age piano music by a wonderful local composer. I have been learning code for the past year or so, and I wanted to use github to do project synchonization and version control. As time went on, I realized that there were some difficulties with using github for that purpose. The command line tools work very well and are very easily incorporated into my workflow, but they are somewhat inaccessible to someone disinterested in spending a good deal of time troubleshooting as I have.
This got me to thinking about a good way to deal with that. On one hand, I could give up on using github for creative work and just make use of some more common file sharing program like Google drive. But, since switching back to Google drive, I find the ability to keep large projects synchonized over multiple devices almost insufferably hard. I think that I could probably figure out a way to automatically update my work from my personal device onto the google drive using a bunch of bash scripting, but I just have not found a way yet. So, eventually I had the idea of creating a simple public website on github pages that runs through some common ways of accessing this wonderful technology!

  • Creating a github account
  • Using github entirely through a web browser
  • Using github through desktop application
  • Using github with the command line on Macos/Linux/Unix
  • Using github with the command line on Windows

Creating a github account

Creating a github account is a very process involving just a few steps.

  1. Navigate to github website or google "github."
  2. Enter whatever email you would like to use
  3. Create a password
  4. Create a username
  5. Tell them that you would not like product updates
  6. Solve a puzzle to tell them that you are human
